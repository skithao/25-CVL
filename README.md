## 🛠️ 环境配置
1. 克隆项目
    ```bash
    git clone git@github.com:skithao/25-CVL.git
    cd 25-CVL
    ```

2. 创建虚拟环境（推荐conda）
    ```bash
    conda create -n 25-CVL python=3.10
    conda activate 25-CVL
    pip install -r requirements.txt
    ```
